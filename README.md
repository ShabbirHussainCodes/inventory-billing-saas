# BillingMars

> Inventory, Billing & AI — all in one orbit.

BillingMars is an AI-powered Inventory & Billing SaaS 
platform for small and medium businesses worldwide.

## Features
- Smart Inventory Management
- GST/VAT Invoice Generation
- AI Profit Intelligence
- Real-time Analytics Dashboard
- WhatsApp Invoice Sharing
- Multi-tenant Architecture
- Global Ready — INR, AED, USD, GBP

## Tech Stack
- Frontend: React.js + Tailwind CSS
- Backend: Django + Django REST Framework
- Database: PostgreSQL
- AI/ML: Prophet + XGBoost
- Payments: Razorpay + Stripe
